# 🏋️ FITZONE Gym Website

A modern and premium gym website built using HTML, CSS, and JavaScript.

This project was created as part of a web development internship project to showcase responsive design, animations, and modern UI concepts.

---

# 🚀 Features

✅ Responsive Design  
✅ Smooth Scrolling Navigation  
✅ Premium Hero Section  
✅ Animated Scroll Reveal Effects  
✅ Glassmorphism Cards  
✅ Membership Pricing Plans  
✅ Trainer Showcase  
✅ Testimonials Section  
✅ Contact Form  
✅ Stylish Hover Effects  
✅ Modern UI/UX Design  

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- ScrollReveal.js
